# Mobile Repair Instant Quote CLI

[![Ruby](https://img.shields.io/badge/Ruby-3.0+-red.svg)](https://www.ruby-lang.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A Ruby command-line tool that provides **instant repair quotes** for mobile phones and iPads. Select brand → model → issue to get accurate pricing instantly.

## ✨ **Features**

- ✅ **Interactive CLI** - Numbered menus for easy navigation
- ✅ **Multi-brand support** - iPhone, Samsung, Google Pixel, Oppo, etc.
- ✅ **Model filtering** - Shows only relevant models per brand
- ✅ **Real-time pricing** - Issue-wise repair costs displayed instantly
- ✅ **Clean interface** - Simple numbered selection system
